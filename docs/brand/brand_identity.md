# 🎨 **BRAND IDENTITY KIT — Concept 1: Minimalist Cinema Icon (Film‑Strip X)**

## **Brand Personality**
- Modern  
- Clean  
- Confident  
- Tech‑forward  
- Streaming‑platform energy  

## **Color Palette**
**Primary**
- Electric Blue — #2D9CFF  
- Charcoal — #1A1A1A  
- Pure White — #FFFFFF  

**Secondary**
- Slate Gray — #2E2E2E  
- Soft Blue — #A7D8FF  

**Accent**
- Neon Aqua — #00F0FF  

## **Typography**
- **Headlines:** Montserrat ExtraBold  
- **Body:** Inter Regular  
- **UI / Labels:** Roboto Medium  
- **Code / Dev Docs:** JetBrains Mono  

## **Iconography**
- Thin‑line icons  
- Rounded corners  
- Geometric shapes  
- Flat, no gradients  

## **UI Style**
- High contrast  
- Lots of negative space  
- Blue accents for active states  
- Rounded buttons, subtle shadows  

## **Brand Voice**
- Direct  
- Clear  
- Helpful  
- Slightly playful but professional  

## **Do / Don’t**
**Do:**  
- Keep spacing wide  
- Use blue sparingly for emphasis  
- Keep the X symbol consistent  

**Don’t:**  
- Add gradients to the wordmark  
- Use serif fonts  
- Overcrowd the layout  

## **Applications**
- **Web header:** Blue underline animation on hover  
- **App UI:** Floating action button with the X icon  
- **Print:** Matte black with blue foil X
