# Install a plugin from the curated catalog

> **Status:** stub. The in-product catalog (a curated list of trusted
> plugins fetched from the Phlex hub) ships in **Phase C** with the
> hub itself. Until then, install plugins by URL — see
> [`install-from-url.md`](install-from-url.md).

When the hub is live, this document will cover:

- Browsing the catalog from `/admin/plugins/catalog`.
- Filtering by plugin type (metadata provider, transcoder, scrobbler, …).
- One-click install via the same JSON API that powers URL installs.
- Signature verification against the hub's published trusted-key list.
